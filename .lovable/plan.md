
# План: Улучшить UX для отображения способов оплаты до подключения кошелька

## Проблема

Сейчас пользователь видит только кнопки выбора сети (ETH/BNB), а способы оплаты (USDT/USDC) скрыты до подключения кошелька. Это создаёт впечатление, что купить токены можно только за нативные монеты.

## Текущий флоу

```text
┌─────────────────────────────────┐
│ 1. Выбор сети: [ETH] [BNB]     │  ← Виден всегда
├─────────────────────────────────┤
│ 2. Способ оплаты: СКРЫТО       │  ← Появляется только после подключения
├─────────────────────────────────┤
│ [ Connect Wallet ]              │
└─────────────────────────────────┘
```

## Предлагаемое решение

```text
┌─────────────────────────────────┐
│ 1. Выбор сети: [ETH] [BNB]     │  ← Виден всегда
├─────────────────────────────────┤
│ 2. Оплата: [ETH] [USDT] [USDC] │  ← ТЕПЕРЬ ВИДЕН ВСЕГДА
├─────────────────────────────────┤
│ [ Connect Wallet to Buy ]       │  ← Чёткий призыв к действию
└─────────────────────────────────┘
```

## Что изменится

### 1. Показать способы оплаты всегда

Убрать условие `{isConnected && (...)}` для блока выбора способа оплаты, чтобы пользователь сразу видел все варианты: ETH, BNB, USDT, USDC.

### 2. Добавить подсказки для неподключённых пользователей

Добавить визуальные подсказки, которые объясняют порядок действий:
- Иконка или лейбл "Step 1: Select Network"
- Иконка или лейбл "Step 2: Choose Payment"
- Кнопка изменится на "Connect Wallet to Buy"

### 3. Добавить бейдж с принимаемыми валютами

В верхней части карточки добавить информационный бейдж:
```text
"Pay with ETH • BNB • USDT • USDC"
```

## Технические изменения

### Файл: `src/components/PresaleCard.tsx`

1. **Удалить условие isConnected** для блока способов оплаты (строка 246)
2. **Добавить лейблы** к секциям выбора сети и оплаты:
   - "Select Network" над кнопками ETH/BNB
   - "Pay With" над кнопками ETH/USDT/USDC
3. **Добавить бейдж** под заголовком: "Accepts: ETH • BNB • USDT • USDC"
4. **Обновить текст кнопки** для неподключённых пользователей: "Connect Wallet to Buy"

## Результат

Пользователь сразу понимает:
- Какие сети поддерживаются
- Что можно оплатить стейблкоинами
- Что для покупки нужно подключить кошелёк
