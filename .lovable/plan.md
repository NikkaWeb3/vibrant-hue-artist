
# План: Значительно расширить карточку покупки токенов

## Проблема
Карточка Presale всё ещё выглядит узкой на широких экранах. Сейчас ограничение `max-w-xl` (576px).

## Решение
Увеличить максимальную ширину карточки и убедиться, что родительский контейнер не ограничивает её размер.

## Технические изменения

### 1. Файл: `src/components/PresaleCard.tsx`

Изменить ширину с `max-w-xl` на `max-w-2xl` (672px) во всех трёх местах:

**Строки 137, 170, 189:**

Было:
```tsx
<div className="glass-card p-6 lg:p-8 w-full max-w-xl">
```

Станет:
```tsx
<div className="glass-card p-6 lg:p-8 w-full max-w-2xl">
```

### 2. Файл: `src/components/HeroSection.tsx`

Изменить контейнер карточки, чтобы он занимал больше места на больших экранах:

**Строка 107:**

Было:
```tsx
<div className="w-full lg:w-auto lg:flex-shrink-0">
```

Станет:
```tsx
<div className="w-full lg:w-[672px] lg:flex-shrink-0">
```

## Размеры для справки
- `max-w-md` = 448px
- `max-w-lg` = 512px  
- `max-w-xl` = 576px (текущий)
- `max-w-2xl` = 672px (новый)

## Результат
- Карточка станет значительно шире (на 96px больше)
- На полноэкранном режиме будет выглядеть пропорционально
- Кнопки, инпуты и таймер получат больше пространства
